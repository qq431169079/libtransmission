Function Documentation

int tr_blocklistSetContent ( tr_session *  session,
const char *  filename 
)

Specify a range of IPs for Transmission to block.

Filename must be an uncompressed ascii file.

libtransmission does not keep a handle to `filename' after this call returns, so the caller is free to keep or delete `filename' as it wishes. libtransmission makes its own copy of the file massaged into a binary format easier to search.

The caller only needs to invoke this when the blocklist has changed.

Passing NULL for a filename will clear the blocklist.
