Returns an fd to the specified filename.

A small pool of open files is kept to avoid the overhead of continually opening and closing the same files when downloading piece data.

  • if do_write is true, subfolders in torrentFile are created if necessary.
  • if do_write is true, the target file is created if necessary.

on success, a file descriptor >= 0 is returned. on failure, a -1 is returned and errno is set.
