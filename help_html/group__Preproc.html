#define lengthof (   arr)    (sizeof (* (arr)) > 0 ? sizeof (arr) / sizeof (* (arr)) : 0)

returns the static length of a C array type

Note
A lower case macro name is tolerable here since this definition of lengthof () is intimately related to sizeof semantics. Meaningful return values are only guaranteed for true array types.
