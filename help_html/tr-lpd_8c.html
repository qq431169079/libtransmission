Detailed Description

Local Peer Discovery.

This module implements the Local Peer Discovery (LPD) protocol as supported by the uTorrent client application. A typical LPD datagram is 119 bytes long.

Initializes Local Peer Discovery for this node.

For the most part, this means setting up an appropriately configured multicast socket and event-based message handling.

Remarks
Since the LPD service does not use another protocol family yet, this code is IPv4 only for the time being.
